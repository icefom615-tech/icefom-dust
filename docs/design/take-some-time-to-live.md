# TAKE SOME TIME TO LIVE — 网站设计说明

## 理解摘要

- 构建一个以作品与 Vlog 为核心的个人展示网站，面向喜欢内容的普通观众。
- 主品牌为 `TAKE SOME TIME TO LIVE`，签名为“花·生 & 影子”。
- 采用高度复古的 2000 年代 Flash 作品集气质，以悬挂物件和实体卡片构成单一互动舞台。
- 核心栏目为 `WORKS`、`VLOG`、`ABOUT US` 与 `CONTACT`。
- 桌面端保留完整舞台；手机端改为纵向悬挂布局，但不移除动画语言。
- Vlog 使用哔哩哔哩嵌入；作品与视频由本地配置文件维护。
- 联系页面仅展示邮箱和社交账号，不收集访客数据。

## 体验设计

首次访问显示独立启动画面。用户点击播放按钮后，Logo、云朵、中央角色与四件导航物件依次由顶部落下，并根据各自重量产生不同幅度的摆动和回弹。中央角色的眼睛追随指针。

选择栏目时，未选择物件上收，让出舞台；选中物件移动到侧边继续摆动；内容卡片沿绳索滑入中央并产生下坠、越界、回弹与阴影变化。返回菜单时播放连续的反向过渡，不瞬间重置。

WORKS 使用悬挂展板和纸张翻页；VLOG 使用旧电视/录像机视觉；ABOUT US 使用日记本结构；CONTACT 使用便签和复古电话。所有栏目共用一套动画状态机，快速连续点击时由新时间轴平滑接管当前状态。

## 非功能要求

- 动画以接近 60fps 为目标，优先使用 transform、opacity 与合成层。
- 首屏不加载视频播放器或全部大图；图片预留尺寸并懒加载。
- 支持现代 Chrome、Safari、Edge 与微信内置浏览器。
- 支持键盘导航、明确焦点，以及系统“减少动态效果”设置。
- 动画或资源失败时仍可浏览内容，并以复古便签显示降级提示。
- 无账户、数据库、评论系统和后台；静态部署，可迁移且维护成本低。

## 决策日志

1. 作品和 Vlog 并重，目标观众为普通内容观众。
2. 采用高度复古 Flash 风格，而不是现代家居杂志风。
3. 英文作为主 Logo，“花·生 & 影子”作为手写签名。
4. 采用单舞台方案，栏目不触发传统页面跳转。
5. 动画是产品核心，必须保留进入、反馈、退出和物理惯性细节。
6. 内容使用配置文件维护；视频由哔哩哔哩承载。
7. 桌面完整、手机重排；联系页不保存访客数据。
8. 使用 React 单舞台架构和统一动画时间轴，首版静态部署。
